Dino World Run now available

Posted on 28 March 2025

#Dino World Run #ProCreate #Godot

After six months of hard work, it’s finally happening: Dino World Run is officially released for Android! 🎉

In this game you play a cute little dino on an epic adventure through hand-drawn worlds. From the overgrown jungle to the scorched volcanic plains and dark caves, many challenging obstacles and lovingly designed levels await you. Challenge yourself and break your own highscore!

The game is made with the Godot game engine. I had a lot of fun learning how to set up and structure a game. As I had never really drawn before, I learned on-the-go how to create assets using ProCreate on an iPad.

For marketing reasons I have created a separate website for the game: dinorun.app. Give it a visit for further details on the game.
