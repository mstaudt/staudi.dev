Spritpreise - Datenschutzerklärung

(Stand: Oktober 2023) Wir freuen uns über Ihr Interesse an unserer App “Spritpreise”. Der Schutz Ihrer Privatsphäre ist für uns sehr wichtig. Von uns erfasste Daten Personenbezogene Daten In keiner unserer Anwendungen oder Dienste werden personenbezogene Daten abgefragt oder übertragen. Beispielsweise werden folgende Daten ausschließlich auf Ihrem Gerät gespeichert und zu keiner Zeit an uns oder Dritte weitergeleitet: Historie der gesuchten Orte und Adressen Als Favorit markierte Tankstellen Verbrauchsdaten Einstellungen der App Standortinformationen Wenn Sie unsere Apps installieren, werden Sie nach Ihrer Einwilligung gefragt, um den Zugriff auf die Standortdaten Ihres mobilen Geräts zu ermöglichen. Wenn Sie eine solche Einwilligung erteilen, erfassen wir Informationen über Ihren Standort und speichern diese Daten ausschließlich auf Ihrem Gerät. Die Standortinformationen werden nur für das Laden von Tankstellen an einen entsprechenden Service weitergeleitet. Sie können Ihre Einwilligung jederzeit widerrufen. Darüber hinaus können Sie die GPS- oder andere Ortungsfunktionen auf Ihrem mobilen Gerät deaktivieren, sofern Ihr mobiles Gerät dies zulässt.
