Dino World Run

Ready for an exciting adventure?

In this game you play a cute little dino on an epic adventure through hand-drawn worlds. From the overgrown jungle to the scorched volcanic plains and dark caves, many challenging obstacles and lovingly designed levels await you. Challenge yourself and break your own highscore!

Dino World Run takes you into a world full of challenges that will put your reflexes to the test!
Please visit dinorun.app for more details.



Tech Stack Link to heading

Godot
Android
iOS
REST-API

This 2D game was made using the Godot game engine. The entire code was written using the engine’s own programming language gdscript, making it platform-independent. This means that both iOS and Android can be supported without much additional work.
In addition, there is a simple REST API that is used to check action codes and associated give-aways.
